# 知识图谱可视化使用说明

## 🎯 功能概述

前端使用 ECharts 实现了交互式知识图谱可视化，支持关键词搜索、节点展开、数据导出等功能。

## 🎨 界面功能

### 搜索区域
- **搜索框**：输入关键词，支持中文
- **搜索历史**：显示最近10条搜索记录，点击可快速重搜
- **重置视图**：清空当前图谱
- **导出数据**：将当前图谱导出为 JSON 文件

### 统计卡片
- **节点数**：当前显示的节点总数
- **关系数**：当前显示的关系总数
- **实体类型**：涉及的实体类型数量

### 图表交互
- **拖拽节点**：可以拖动节点调整布局
- **滚轮缩放**：鼠标滚轮放大/缩小
- **点击节点**：展开该节点的关联节点
- **悬停提示**：鼠标悬停显示详细信息
- **图例筛选**：点击图例隐藏/显示某类节点

## ⚠️ 常见问题

### Q1: 搜索无结果？
**可能原因**：
- 关键词拼写错误
- 数据库中不存在该实体

**解决方法**：
- 检查拼写
- 尝试其他相关关键词
- 查看搜索历史中的可用关键词

### Q2: 图表加载慢？
**可能原因**：
- 节点数量过多（>200个）
- 浏览器性能限制

**解决方法**：
- 刷新页面重新开始
- 减少展开的节点数量

### Q3: 节点重叠严重？
**解决方法**：
- 拖动节点手动调整
- 使用滚轮缩小视图
- 点击"重置视图"重新布局

### Q4: 后端连接失败？
**检查**：
```bash
# 确认后端是否运行
curl http://localhost:8000/api/kg/stats

# 确认 Neo4j 是否运行
docker ps | grep neo4j
```

## 🔧 技术实现

### 前端技术栈
- **Vue 3**: 组件化开发
- **Element Plus**: UI 组件库
- **ECharts 5**: 图表可视化
- **Axios**: HTTP 请求

### 关键代码
```javascript
// 初始化 ECharts
this.chart = echarts.init(this.$refs.chartContainer)

// 配置力导向图
{
  type: 'graph',
  layout: 'force',
  force: {
    repulsion: 800,
    edgeLength: [100, 250]
  }
}

// 颜色映射
colorMap: {
  'Disease': '#FF6B6B',
  'Symptom': '#4ECDC4',
  // ...
}
```

### 后端 API
```python
# 查询实体及关联
GET /api/kg/entities/{entity_name}

# 返回格式
{
  "results": [
    {
      "n": {"name": "胃痛", "category": "Disease"},
      "m": {"name": "胃脘疼痛", "category": "Symptom"},
      "r": {"type": "HAS_SYMPTOM"}
    }
  ]
}
```
